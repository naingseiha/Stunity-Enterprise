## üöÄ Getting Started

### 1. Clone Repository
```bash
git clone https://github.com/yourusername/stunity.git
cd stunity
```

### 2. Install Dependencies
```bash
npm install
```

### 3. Set Up Environment
```bash
cp .env.example .env.local
# Edit .env.local with your credentials
```

### 4. Set Up Database
```bash
npx prisma generate
npx prisma db push
npx prisma db seed
```

### 5. Run Development Server
```bash
npm run dev
```

### 6. Open Browser
```
http://localhost:3000
```
