# Debug Scripts

These scripts are ad hoc inspection helpers used to verify data state, migration outcomes, and edge cases during debugging.

Current files:

- `check-merge.ts`
- `check-teacher-class.ts`
- `check-v1-null.ts`
- `check_data.ts`
- `count-ay-detail.ts`
- `count-ays.ts`
- `count-sc.ts`
- `find-rn-text-error.js`

They are intentionally separated from production entry points and from the main manual test scripts.
